Guide d'utilisation

Découvrez comment utiliser SpeakFree pour signaler en toute sécurité

Comment faire un signalement ?

Vous avez deux façons de signaler un incident sur SpeakFree :

Option 1 : Chat IA Haniel (Recommandé)

Le moyen le plus simple et le plus guidé pour faire un signalement.

  1. Accédez au Chat IA Haniel

    Depuis la page d'accueil, cliquez sur "Parler à Haniel IA" ou accédez directement à la page de chat.

  2. Haniel vous guide étape par étape

    L'IA vous posera des questions simples pour comprendre votre situation :

    • Sélection de votre établissement
    • Type d'incident (harcèlement, violence, discrimination, etc.)
    • Date approximative de l'incident
    • Lieu de l'incident
    • Description détaillée
    • Présence de témoins

  3. Confirmez votre signalement

    Haniel vous montrera un récapitulatif de votre signalement. Confirmez pour l'envoyer.

  4. Recevez vos codes

    Vous recevrez automatiquement :

    • Code de suivi (RPT-XXXXX) : Pour suivre l'avancement
    • Code de discussion (DSC-XXXXX) : Pour communiquer avec l'école

Astuce

Vous pouvez reprendre une conversation avec Haniel à tout moment depuis la page d'accueil en cliquant sur "Reprendre conversation Haniel".

Option 2 : Formulaire de signalement

Pour ceux qui préfèrent remplir un formulaire traditionnel.

  1. Choisissez votre établissement

    Sur la page d'accueil, sélectionnez votre école dans la liste des établissements partenaires.

  2. Remplissez le formulaire

    Décrivez l'incident en détail : type, date, lieu, description, témoins éventuels.

  3. Soumettez votre signalement

    Cliquez sur le bouton d'envoi. Votre signalement sera transmis de manière anonyme.

  4. Conservez vos codes

    Notez précieusement les codes de suivi et de discussion qui vous sont fournis.

Communiquer avec l'établissement

  1. Accédez à l'espace discussion

    Depuis la page d'accueil, cliquez sur "Discuter avec l'école".

  2. Entrez votre code de discussion

    Utilisez le code DSC-XXXXX que vous avez reçu lors de votre signalement.

  3. Échangez en toute confidentialité

    Vous pouvez envoyer des messages et recevoir des réponses de l'administration de votre établissement, tout en restant anonyme.

Important

L'établissement ne connaît pas votre identité. Seul le contenu de vos messages est visible. Vous restez 100% anonyme.

Inscription d'un établissement

  1. Accédez au formulaire d'inscription

    Cliquez sur "Inscrire mon établissement" depuis la page d'accueil ou la page des établissements.

  2. Remplissez les informations

    Nom de l'établissement, adresse email officielle, numéro de téléphone et mot de passe.

  3. Attendez la validation

    Votre demande sera examinée par le super-administrateur SpeakFree. Vous recevrez une confirmation une fois approuvé.

Note

Seuls les établissements scolaires officiels peuvent s'inscrire. Les demandes sont vérifiées manuellement.

Confirmer la résolution d'un signalement

  1. Ouvrez le signalement

    Cliquez sur un signalement pour voir ses détails.

  2. Traitez l'incident

    Communiquez avec l'élève si nécessaire et prenez les mesures appropriées.

  3. Cliquez sur "Confirmer la résolution"

    Le bouton "Confirmer la résolution" dans les détails du signalement permet de le marquer comme résolu.

Résultat

Le statut du signalement passera à "Résolu" et l'élève pourra le voir dans son suivi.

Suivre mon signalement

Restez informé de l'avancement de votre signalement grâce à votre code de suivi.

  1. Accédez au suivi

    Depuis la page d'accueil, cliquez sur la carte "Suivre mon Signalement" ou sur le bouton dédié.

  2. Entrez votre code de suivi

    Saisissez le code au format RPT-XXXXX que vous avez reçu lors de votre signalement.

  3. Consultez le statut

    Vous verrez l'état actuel de votre signalement :

En attente

Votre signalement a été reçu et attend d'être examiné par l'administration.

En cours de traitement

L'établissement examine votre signalement et prend des mesures.

Résolu

L'incident a été traité et le signalement est clôturé.

Conservez vos codes !

Notez précieusement vos codes de suivi et de discussion. Ce sont les seuls moyens d'accéder à votre signalement et de communiquer avec l'école.

Exemple de code de suivi :

RPT-12345

Exemple de code de discussion :

DSC-67890

Questions fréquentes

Mon signalement est-il vraiment anonyme ?

Oui, absolument. SpeakFree ne collecte aucune information permettant de vous identifier (pas d'inscription, pas d'email, pas d'adresse IP stockée). Seul le contenu de votre signalement est transmis à l'établissement.

Comment savoir si mon signalement a été pris en compte ?

Utilisez votre code de suivi (RPT-XXXXX) sur la page d'accueil dans la section "Suivre mon Signalement". Vous pourrez voir le statut actuel : En attente, En cours de traitement, ou Résolu.

Puis-je signaler un incident qui s'est passé il y a longtemps ?

Oui, vous pouvez signaler tout incident, quelle que soit la date. Indiquez simplement la date approximative lors de votre signalement.

Que faire si j'ai perdu mes codes ?

Malheureusement, pour garantir l'anonymat, nous ne pouvons pas récupérer vos codes. C'est pourquoi il est essentiel de les noter et les conserver précieusement dès leur réception.

Comment fonctionne le Chat IA Haniel ?

Haniel est un assistant IA qui vous guide étape par étape pour créer votre signalement. Il vous pose des questions simples (établissement, type d'incident, date, lieu, description) et génère automatiquement votre signalement. C'est le moyen le plus simple et le plus rapide de signaler.

Puis-je reprendre une conversation avec Haniel ?

Oui ! Depuis la page d'accueil, cliquez sur "Reprendre conversation Haniel" et entrez l'ID de session que vous avez reçu. Vous retrouverez votre conversation là où vous l'avez laissée.

Mon établissement n'est pas dans la liste, que faire ?

Vous pouvez suggérer à l'administration de votre établissement de s'inscrire sur SpeakFree. L'inscription est gratuite et permet aux élèves de signaler en toute sécurité.

Que se passe-t-il après la résolution de mon signalement ?

Une fois que l'établissement a traité votre signalement et cliqué sur "Confirmer la résolution", le statut de votre signalement passe à "Résolu". Vous pouvez le voir en utilisant votre code de suivi.
