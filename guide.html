📝 Comment Faire un Signalement

1

Choisir votre École

Trois options disponibles :

  • Code d'école : Saisissez le code fourni par votre administration (ex: LYC1234)
  • Liste existante : Sélectionnez dans la liste des écoles enregistrées
  • Saisie libre : Tapez le nom de votre école (elle sera créée automatiquement)
2

Sélectionner le Mode

Deux modes de signalement :

  • Anonyme : Votre identité reste totalement secrète
  • Avec compte : Vous pouvez suivre facilement vos signalements
3

Remplir le Formulaire

Informations requises :

  • Type d'utilisateur : Élève, Professeur, Parent, Personnel, Autre
  • Catégorie : Harcèlement, Violence, Discrimination, etc.
  • Niveau d'urgence : Faible, Modérée, Élevée, Critique
  • Titre : Résumé court du problème
  • Description : Détails complets de la situation
4

Informations Complémentaires

Optionnel mais utile :

  • Lieu : Où s'est passé l'incident
  • Date : Quand cela s'est produit
  • Témoins : Y a-t-il eu des témoins
  • Première fois : Est-ce la première occurrence
5

Envoyer et Suivre

Après envoi, vous recevrez un code de suivi unique :

Exemple : SF-1729123456-ABC12

Conservez ce code pour suivre l'évolution de votre signalement.

🏫 Pour les Administrations d'École

📋 Inscription de votre Établissement

  1. Rendez-vous sur la page d'inscription école
  2. Remplissez toutes les informations de votre établissement
  3. Votre école et votre compte administrateur sont créés immédiatement
  4. Vous recevez un code d'école unique (ex: LYC1234)
  5. Communiquez ce code à vos élèves, professeurs et parents

🔑 Gestion de votre École

Avec votre compte administrateur, vous pouvez :

  • Voir tous les signalements de votre établissement
  • Répondre de manière anonyme aux signaleurs
  • Gérer les utilisateurs de votre école
  • Générer des liens d'invitation pour vos membres
  • Consulter les statistiques détaillées
  • Exporter les données pour analyse

👥 Création de Compte Utilisateur

🆕 Inscription Normale

  1. Cliquez sur "Inscription" depuis la page d'accueil
  2. Choisissez votre mode de sélection d'école
  3. Remplissez vos informations personnelles
  4. Créez votre compte
  5. Connectez-vous pour accéder à vos fonctionnalités

📧 Inscription par Invitation

  1. Cliquez sur le lien d'invitation de votre administration
  2. Votre école est automatiquement pré-sélectionnée
  3. Remplissez vos informations
  4. Votre compte est lié à votre établissement

📊 Suivi de vos Signalements

🔍 Avec votre Code de Suivi

Utilisez la section "Suivre" pour :

  • Voir le statut actuel de votre signalement
  • Lire les réponses de l'administration
  • Suivre l'évolution du traitement
  • Recevoir des mises à jour

👤 Avec votre Compte Personnel

Dans votre profil, vous pouvez :

  • Voir tous vos signalements en un coup d'œil
  • Accéder rapidement aux détails
  • Recevoir des notifications
  • Gérer vos préférences

🛡️ Sécurité et Anonymat

🔒 Protection de votre Anonymat

  • Aucune donnée personnelle n'est collectée pour les signalements anonymes
  • Chiffrement de toutes les communications
  • Codes de suivi impossibles à relier à votre identité
  • Serveurs sécurisés protégeant vos données

⚠️ Cas Exceptionnels

Dans des situations très graves (crimes, danger imminent, déstabilisation), l'administration peut demander la révélation de l'identité, conformément à la loi ivoirienne.

❓ Questions Fréquentes

Q: Mes signalements sont-ils vraiment anonymes ?

R: Oui, pour les signalements anonymes, aucune information permettant de vous identifier n'est collectée. Seul le contenu de votre signalement est transmis à l'administration.

Q: Que se passe-t-il après mon signalement ?

R: Votre signalement est immédiatement transmis à l'administration de votre école. Vous recevez un code de suivi pour suivre l'évolution du traitement.

Q: Puis-je modifier mon signalement après l'avoir envoyé ?

R: Non, les signalements ne peuvent pas être modifiés après envoi. Cependant, vous pouvez faire un nouveau signalement avec des informations complémentaires.

Q: Mon école n'est pas dans la liste, que faire ?

R: Choisissez "Saisir le nom de mon école" et tapez le nom de votre établissement. Il sera automatiquement créé dans le système.

Q: Combien de temps pour avoir une réponse ?

R: Cela dépend de votre école et de la gravité du signalement. Les cas urgents sont traités en priorité. Vous pouvez suivre l'évolution avec votre code.
